# Esme
A fork of Minwaita which is in turn a fork of Adwaita Compact. Unlike Minwaita, this theme is more liberal when it comes to design tweaks and spacing. The goal with this one is a clean, futuristic look with some homage to Gaia (which seems to be dead?).
